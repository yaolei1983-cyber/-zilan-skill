## §1 核心命题：缘起 = 无自性 = 空性

### 根本颂

> 《中论》第二十四品 · 第十八颂：
> **"众因缘生法，我说即是空，亦为是假名，亦是中道义。"**

### 三等式

```
缘起（pratītyasamutpāda）
  = 无自性（niḥsvabhāva）
  = 空性（śūnyatā）
  = 假名（prajñapti）
  = 中道（madhyamā pratipad）
```

### 五个等号的含义

| 等式 | 逻辑 | 含义 |
|------|------|------|
| 缘起 = 无自性 | 凡依赖因缘而生者，必无独立自存的自性 | 摧破自性见 |
| 无自性 = 空性 | "空"不是虚无，而是"自性空"——无自性即是空 | 区分空性与断灭 |
| 空性 = 假名 | 正因为自性空，才能安立名言/概念/作用 | **"性空故缘起"——空性是缘起的可能性条件** |
| 假名 = 中道 | 既非常见（有自性），亦非断见（全无） | 远离二边 |

### 月称的"三义缘起"

月称在《入中论》中对"缘起"做了字源学深化：

```
缘（pratītya）= 到达 (prāpti) = 相待 (apekṣā)
起（samutpāda）= 生起 (utpāda)

→ 缘起 = 相待而生
→ 三义：
  ① 此缘性（conditionality）— "此有故彼有，此生故彼生"
  ② 相待性（relativity）— 长待短、左待右、自待他
  ③ 空寂性（emptiness）— 正因为相待，所以无独立自性
```

### 类比映射

```
自性（svabhāva / rang bzhin）
  = 不依赖任何因缘、独立自存的本质属性
  = 类比：hardcoded constant（硬编码常量）

无自性（niḥsvabhāva）
  = 不存在独立自存的本质属性，一切依缘而起
  = 类比：runtime dependency injection（运行时依赖注入）

空性（śūnyatā）
  = 无自性这一事实本身
  = ≠ null / undefined（不是"没有"）
  = 类比：absence of hardcoded constants — the system is fully dynamic
```

## §2 应成论式作为中观方法论

### 方法论宣言

应成派的核心方法论立场：**不立自宗，唯破他执。**

```
自续派（Svātantrika）：
  "我们用自立的因和喻来论证空性。"
  → 问题：若因和喻有自性，则与"无自性"自相矛盾
  → 若因和喻无自性，则对方若已接受无自性，无需论证；
    对方若不接受无自性，不共许的因和喻如何成立？

应成派（Prāsaṅgika）：
  "我们不立任何自宗。我们只指出对方论题的内在矛盾。
   当对方说'X有自性'时，我们只说：若X有自性，则应成……（矛盾）'。
   对方自己的前提自然会推出荒谬结论——不需要我们另建前提。"
```

### 应成论式的中观用法

```
摄类学 §6 定义的应成论式 = ⟨有法，应成法，因⟩

中观语境下的用法：

  ⟨X⟩，应是⟨无自性⟩，以⟨是缘起法⟩故。

  对方（执实有者）：
    若 X 有自性，则 X 不能是缘起法（有自性 = 不依赖因缘）
    但 X 确实是缘起法（因缘和合而生）
    → 矛盾。故 X 不能有自性。

  中观师不立"X 无自性"作为独立命题——
  只归谬：有自性 → 非缘起 → 与事实矛盾 → 故有自性不成立。
```

### 与摄类学 §6 标准应成论式的区别

| 维度 | 摄类学标准应成式 | 中观应成式 |
|------|----------------|-----------|
| **目的** | 检验任一命题的逻辑有效性 | 归谬他宗以显空性 |
| **因的来源** | 可以是共许或客观事实 | 取自对方的前提 |
| **结论状态** | 承许 / 因不成 / 不周遍 | 对方前提 → 自相矛盾 |
| **是否立宗** | 可立可破 | 只破不立 |

## §3 二谛论纲要

### 二谛定义

```
世俗谛（saṃvṛti-satya / 覆俗谛）：
  凡夫被无明所覆而见为实有的现象世界
  "覆"的三义：覆障真实、相互依存、世间言说

胜义谛（paramārtha-satya）：
  圣者在定中以无分别智亲证的真实——一切法无自性、毕竟空
```

### 月称的世俗谛立场

月称在二谛上的立场比自续派（清辨）更彻底：

| 自续派 | 应成派（月称） |
|--------|-------------|
| 世俗中可承许有自相（svalakṣaṇa） | 世俗中亦不承许有自相 |
| "世俗谛有自相，胜义谛无自性" | "二谛皆无自性" |
| 世俗谛的"自相"是世俗共许的事实 | "自相"本身就是无明所执——即使在世俗中也找不到 |

**月称的世俗谛**：不承许任何"自相有"，世俗谛就是**无明所覆的凡夫所见**。这不是说世俗中有些东西有自相——世俗谛的定义本身就以无明为前提。

### 类比

```
胜义谛 = source code（实际运行的是什么）
世俗谛 = user interface（用户看到的是什么）

月称的立场：
  不是"UI 上这个按钮有颜色"（自相有）而"代码里按钮只是像素"（空）
  而是：按钮在 UI 上呈现为"有颜色的实体"本身就是一种误认——
  即使在 UI 层面，也没有一个"按钮自性"。
  UI 的全部呈现都是依赖代码/硬件/电源/用户的缘起。
```

## §4 核心论证选例

### 4.1 破自生、他生、共生、无因生（《中论》第一品）

龙树以四句逻辑穷尽"生"的所有可能，逐一归谬：

```
问题：果法从何而生？

句1：自生（果从自因生）—— 数论派主张
  → 若果已存于因中，则生是多余的（已有何必生？）
  → 若果不存于因中，则"自生"不成立（不是同一物如何自生？）
  → 破

句2：他生（果从他因生）—— 胜论派/经部等主张
  → 若因与果是"他性"（互为他者），则一切能从一切生（火生黑暗？）
  → 若必须是特定的"他"，则这个特定关系本身 ≠ 纯粹的"他性"
  → 破

句3：共生（果从自他共生）—— 耆那教主张
  → 自生已破、他生已破 → 共生（自+他）亦不成立
  → 破

句4：无因生（果无因而生）—— 顺世派主张
  → 若一切无因，则一切恒常或恒无（无法解释季节、因果等世间规律）
  → 破

结论：四句皆破 → "生"本身是假名安立，没有自性的"生"。
     这恰好证明了"缘起 = 无自性"。
```

### 4.2 破"去来"（《中论》第二品）

```
问题：有"去"这个动作吗？

⟨去者⟩，应不能⟨去⟩，以——
  · 若"去者"在去之前已经存在，则"去"是多余的（已去者何必再去？）
  · 若"去者"依赖"去"才成立，则"去者"与"去"互为前提——循环
  · 若"去者"与"去"同时成立，则没有"去者去"的先后关系

→ 去者、去法、去处——三者皆无自性。
  日常生活中的"张三走去公司"是缘起的假名安立，不是自性有。
```

### 4.3 金刚屑因（破自生之胜义因）

月称在《入中论》第六品中，以金刚屑因集中破自生：

```
金刚屑因 = ⟨一切法有自性的生⟩ 是不可能的，以——

  若生有自性，则必是四者之一：自生、他生、共生、无因生。
  四者皆不可能（§4.1 已破）。
  故"有自性的生"不成立。
  故"生"唯是缘起假名。

此因属于法称三因中的"不可得因"：
  若"有自性的生"存在，以正理可得；实际不可得；故"有自性的生"不存在。
```

## §5 破唯识四论（《入中论》第六品）

月称在第六品系统破斥唯识宗四大核心学说。这是应成派与唯识派的关键分水岭。

### 5.1 破阿赖耶识

```
唯识主张：必须有阿赖耶识作为业种存储之所依，否则业力无法相续。

月称反破：
  · 业力不需要一个"实有的存储容器"。
  · 业力的相续机制：现行（显）→ 消灭（隐）→ 遇缘成熟（显）——
    就像种子不需要一个"种子柜"，种子自身的相续就够了。
  · 安立阿赖耶识 = 对"业的相续"做了不必要的实体化。
```

### 5.2 破唯识无境

```
唯识主张：外境不存在，唯识所现（以梦喻、眩翳喻为证）。

月称反破：
  · 梦喻不能成立"唯识无境"：
    若梦中无境，则梦中的"识"也应是假（心境相待不离）。
    若梦中的"识"是假，则以此喻证唯识——喻本身不成立。
  · 眩翳喻不能成立"唯识无境"：
    翳者见毛——若毛纯属识的变现而无外境，则"翳者的眼根"也应是无外境的幻相。
    根与境同有同无：根若无，识依何起？
  · 结论：心境相待而立——没有无境之心，也没有无心之境。
    但这不是说"心外有实境"——而是说"境"和"心"都是缘起假名。
```

### 5.3 破自证分

```
唯识主张：认知能自证——心识能认知自身（如灯自照、刀不自割之辩）。

月称反破：
  · 刀不自割、手不自触 —— 能所不能同一。
  · 记忆不需要自证分：我记得"昨天见了蓝色"，不需要假定"当时有一个自证分
    认知了'我见了蓝色'"——因为记忆可以缘过去境。
  · 自证分的真实动机是确立"识的实有"——但识也是缘起、无自性。
```

### 5.4 破依他起有自性

```
唯识主张：依他起性（因缘所生法）有自性——圆成实性的空性依于依他起的实有。

月称反破：
  · 这是唯识与中观的根本分歧。
  · 依他起 = 因缘所生。因缘所生 = 无自性（§1 的三等式）。
  · "因缘所生法"与"有自性"是逻辑矛盾——不可兼得。
  · 唯识说"他性空"（依他起上无遍计所执）还不够——
    必须是"自性空"（依他起本身也无自性）。
```

### 5.5 宗喀巴总结的"八难题"

宗喀巴在《入中论善显密意疏》中将应成派的独特主张概括为：

| 范畴 | 应成派的独特立场 |
|------|-----------------|
| **名言中不许** | ❌ 不许有自相、❌ 不许阿赖耶识、❌ 不许自证分、❌ 不许自续因 |
| **名言中许** | ✅ 许有外境、✅ 许二障之安立、✅ 许声闻独觉亦通达法无我 |
| **果位** | ✅ 许佛尽所有智 |

## §6 与摄类学/因明/心类学工具的对接

### 工具调用链

```
讨论佛教哲学命题
    │
    ├── 涉及概念分析
    │   → 摄类学 §1 性相与所表（给"空性""自性""缘起"下精确定义）
    │   → 摄类学 §2 总与别（空性与十八空、十六空的总别层级）
    │   → 摄类学 §5 四句逻辑（破四生的四句穷尽）
    │
    ├── 涉及推理验证
    │   → 因明 §1 因三相（检验"金刚屑因"是否满足三相）
    │   → 因明 §2 三因说（归谬论证 → 不可得因的否定推理模式）
    │   → 摄类学 §7 破与立（只破不立 → 无遮遮止）
    │
    ├── 涉及认知状态分析
    │   → 心类学 §3 七种认知（"执实有自性"是哪种认知？→ 颠倒知）
    │   → 心类学 §4 转化次第（从执实到证空的心识转化路径）
    │
    └── 涉及实修应用
        → 摄类学 §6 应成论式 Debug（三大长间隙的日常归谬）
        → 南传观禅指南（从推论到现观的止观路径）
```

### 中观不共的工具使用协议

```
中观应成派的论证 = 摄类学 §7（破与立）+ 因明 §2（不可得因）

具体流程：
  1. 对方声称 X 有自性
  2. 从对方的前提中推导出一个矛盾（应成论式）
  3. 使用不可得因：若 X 有自性，以正理可得；实际不可得 → X 无自性
  4. 使用无遮遮止：只说"X 无自性"，不立"X 是什么"的正面替代
  5. 对方自证矛盾 — 中观师不另立宗

"破而不立"不是"什么都不说"——
而是严格限定在"归谬他宗"的操作中，不建立自己的正相命题。
```
