>
> 本文件是孜澜的认知结构分析基础设施。加载时机：当讨论涉及心识运作机制、烦恼生起与对治、认知正确性判定、"谁在认知"的主体性分析，或需要将心识活动转化为结构化 Debug 对象时。

## §1 心王 —— 认知的操作系统进程

### 定义

```
心（sems / citta）= "明而了别"
  · 明（gsal）：心本性清净光明，如镜映现
  · 了别（rig pa）：心能主动认知、辨别对境

心王（gtso sems / primary mind）= 认知活动的主导者，如操作系统主进程
```

### 六识分类

心王按所依根与所缘境分为六种——类比六个 I/O 通道：

| 识 | 所依根 | 所缘境 | 类比 |
|----|--------|--------|------|
| **眼识** | 眼根 | 色处（颜色、形状） | Video input |
| **耳识** | 耳根 | 声处（声音） | Audio input |
| **鼻识** | 鼻根 | 香处（气味） | Chemical sensor 1 |
| **舌识** | 舌根 | 味处（味道） | Chemical sensor 2 |
| **身识** | 身根 | 触处（触觉） | Tactile input |
| **意识** | 意根 | 法处（一切法） | CPU — 处理所有输入 |

### 根识与意识的分工

```
根识（前五）：
  · 只缘现在（不能缘过去/未来）
  · 只缘外境（物质性的）
  · 离分别（不涉及概念判断）
  · 只在自之所缘境上运作

意识：
  · 可缘三世（过去、现在、未来）
  · 可缘内外（物质性 + 非物质性，包括概念、记忆、情绪）
  · 可分别（涉及概念、判断、推理）
  · 可在一切境上运作
```

### 实修关键

日常掉线的起点通常在**意识层**：前五根识忠实地传递了原始数据（领导说了某句话），但意识对这些数据进行了**遍计所执性的二次加工**（这句话 = 对我价值的否定）。

这就是为什么"实时Debug心识流"的主战场在意识。

## §2 心所 —— 认知的系统调用

### 定义

```
心所（sems byung / mental factor）= 伴随心王生起的心理作用
类比：心王是进程，心所是该进程调用的函数库
```

### 心王与心所的五相相应

| 相应维度 | 含义 | 类比 |
|---------|------|------|
| **所依相应** | 依赖同一个根 | 同一个硬件 |
| **所缘相应** | 缘取同一个对象 | 同一个 input |
| **行相相应** | 显现同一个行相 | 同一个内存表示 |
| **时相应** | 同时生起 | 同一个时钟周期 |
| **实质相应** | 同一实质生起 | 同一个进程空间 |

### 五十一心所速查

#### 五遍行（与一切心王伴随）

| 心所 | 功能 | 类比 |
|------|------|------|
| **触** | 根·境·识三者和合 | `connect()` — 建立感知连接 |
| **作意** | 令心朝向所缘 | `focus()` — 注意力分配 |
| **受** | 领纳苦·乐·舍 | `eval()` — 情感标记 |
| **想** | 取相安立名言 | `classify()` — 特征提取+命名 |
| **思** | 令心造作、发动业 | `exec()` — 意志执行 |

> **五遍行是认知的最小运行单元。** 任何一次认知都必然包含这五个环节——从触到思是一条不可分割的流水线。

#### 五别境（特定认知场景才出现）

| 心所 | 功能 | 触发条件 |
|------|------|---------|
| **欲** | 希望、欲求 | 对可爱境 |
| **胜解** | 确定不移的认可 | 对已决断之事 |
| **念** | 不忘失、明记 | 对熟悉境 |
| **定（三摩地）** | 心一境性 | 对所观境 |
| **慧** | 抉择、简择诸法 | 对需抉择之境 |

#### 善心所（11 种）—— 调试工具集

| 心所 | 功能 | 三大长间隙中的角色 |
|------|------|-------------------|
| **信** | 相信真实、有功德、有能力者 | 工作目的参照系不为他人评价所动的锚 |
| **惭** | 基于自我尊重而拒恶 | "我作为修行者不该这样反应" |
| **愧** | 基于他人尊重而拒恶 | "这样对待家人不合理" |
| **无贪** | 不执著 | 不将"在乎"绑定"应得回报" |
| **无瞋** | 不愤怒、慈心 | 孩子哭闹时不爆发 |
| **无痴** | 明辨因果、如实知 | 看清"被否定 ≠ 无价值" |
| **精进** | 断恶修善的勇悍 | 持续修行的动力 |
| **轻安** | 身心调柔轻快 | 压力下的生理调节 |
| **不放逸** | 守护心不令散乱 | 觉察间隙的缩短 |
| **行舍** | 心平等、不偏执 | 两性计较时的泰然 |
| **不害** | 不忍伤害众生 | 对孩子的耐心 |

#### 根本烦恼（6 种）—— 核心 Bug 集

```
贪（rāga）──── 执著、欲求不舍 ──── 映射：内存泄漏（资源不放）
瞋（dveṣa）─── 排斥、愤怒、厌恶 ── 映射：进程崩溃（强制终止）
慢（māna）──── 自恃、比较心 ────── 映射：类型膨胀（自我类型过大）
无明（avidyā）─ 不知业果、不知谛理 ─ 映射：核心逻辑错误
染污疑（vicikitsā）─ 犹豫不决 ──── 映射：死循环（无法决策）
染污见（dṛṣṭi）── 错误认知框架 ─── 映射：错误的类型系统
```

#### 随烦恼（20 种）—— 从根本烦恼衍生的子 Bug

聚焦与三大长间隙直接相关的：

| 随烦恼 | 触发场景 | 表现 |
|--------|---------|------|
| **忿** | 职场否定 | 立即的情绪爆发 |
| **恨** | 职场否定（持续） | 长时间的不满淤积 |
| **嫉** | 两性计较 | "她凭什么不在乎我" |
| **恼** | 育儿溃败 | 烦躁不安 |
| **昏沉** | 高强度工作后 | 认知能力下降 |
| **掉举** | 碎片化时间 | 心念散乱无法聚焦 |
| **不信** | 三大场景通用 | 对法义的临时丧失 |
| **懈怠** | 三大场景通用 | "反正修了也没用" |
| **失念** | 三大场景通用 | 忘记修行本心 |
| **不正知** | 三大场景通用 | 对当下心识状态无知 |

#### 四不定（可与善/恶/无记任一组合）

| 心所 | 说明 |
|------|------|
| **睡眠** | 心识昏昧 — 非善非恶，但过长则障碍修行 |
| **恶作（悔）** | 对已作/未作之事的后悔 — 可为善（悔未修善）可为恶（悔已修善） |
| **寻** | 粗分别心 — 心念初步趋向所缘 |
| **伺** | 细分别心 — 心念精细审查所缘 |

## §3 量与非量 —— 七种认知分类（核心协议）

### 定义

```
量（tshad ma / pramāṇa）= 新起、不欺诳之了别
  · 新起：对彼境而言是初次认知（不是回忆/再认）
  · 不欺诳：对该境的认知符合实际情况
```

### 七种认知完整分类

```
一切认知
├── 量知（正智 / 正确认知）—— 2 种
│   ├── 现量（pratyakṣa / 直接感知）
│   │   ├── 根现量 — 五根的离分别直接感知
│   │   ├── 意现量 — 意识对前五识的直接觉知
│   │   ├── 自证现量 — 认知对自身的觉知（经部/唯识许，应成派不许）
│   │   └── 瑜伽现量 — 修行者定中对空性/无我的直接现观
│   └── 比量（anumāṇa / 推理认知）
│       ├── 物力比量 — 基于客观事实的推理（如：声无常，所作性故）
│       ├── 世许比量 — 基于世间共许的推理（如：见烟知火）
│       └── 信仰比量 — 基于经教清净的推理（如：施得财富）
│
└── 非量知（非正智 / 错误认知）—— 5 种
    ├── 已决智（再决知） — 不是新证，而是对已证之境的再次了知
    ├── 伺察意 — 未经理证仅凭言词就完全相信（"我觉得空性大概就是这样"）
    ├── 显而未定知 — 境虽显现但未能确认（走神时眼前画面存在但未注意）
    ├── 犹豫识（疑） — 对境两边摇摆（"是无常还是常？不确定"）
    └── 颠倒知 — 对境完全颠倒（执无我为有我，执无常为常）
```

### 七种认知的效力判定

| 认知类型 | 能否真正了解对境 | 说明 |
|---------|:---:|------|
| 现量 | ✅ | 离分别、不错乱、新证 |
| 比量 | ✅ | 依正因推理、新证 |
| 已决智 | ✅ | 已了解后的再确认（非新证） |
| 伺察意 | ❌ | 信而未证 |
| 显而未定 | ❌ | 现而未证 |
| 犹豫识 | ❌ | 未决断 |
| 颠倒知 | ❌ | 完全错误 |

**关键**：只有现量和比量能**新证**对境。已决智只是对已证境的再次确认。

### 实修含义

```
日常的心识 Debug 问题本质上是认知质量判定问题：

"被领导否定 = 工作无价值"
  → 这是什么类型的认知？
  → 不是现量（不是直接感知到的客观事实）
  → 不是正比量（没有正因：领导否定 ⊆ 工作无价值？→ 不周遍）
  → 是颠倒知（将"一个人的单次评价"等同于"客观价值"——遍计所执性）

"孩子在故意气我"
  → 这是什么类型的认知？
  → 11 个月婴儿的哭闹 = 生理需求表达（这是正确的认知）
  → "故意气我" = 对婴儿行为动机的颠倒知（婴儿尚无"故意气人"的心智能力）
```

## §4 认知转化次第 —— 从颠倒到现量的 Debug 路径

### 完整转化链路

```
颠倒知（我执、实执、常执）
    │  【听闻正法、思惟法义】
    ▼
犹豫识（不符合事实的犹豫 → 等分犹豫 → 符合事实的犹豫）
    │  【持续听闻 + 如理思惟 + 正因推理】
    ▼
伺察意（相信正确结论但未经正因证成）
    │  【学习并使用因明推理（因三相）建立正比量】
    ▼
物力比量（以正因新证正确结论的决定解）
    │  【反复串习比量 → 再决知 → 比量 → 再决知……】
    ▼
比度再决知（第二刹那及之后的比量确认）
    │  【修持止观双运三摩地】
    ▼
瑜伽现量（现证空性/无我的圣智）
    │  【圣道生起，正对治我执】
    ▼
现前再决知（现量第二刹那之后）
```

### 当前修学阶段的映射

根据 SKILL.md 用户画像：
- **"觉察间隙：大多数时候在事情发生后才觉察到'掉线'"**
  → 认知状态：介于颠倒知与犹豫识之间。掉线时是颠倒知，事后觉察是犹豫识（"我刚才的反应对吗？"）

- **目标**：将觉察从"事后"移到"正在发生"（甚至"发生之前"）
  → 认知转化：颠倒知 → 犹豫识（缩短）→ 伺察意（建立正确认知框架）→ 物力比量（在日常中用正因实时分析）

## §5 三大长间隙的心识 Debug

### 场景 A：职场否定

```
触发序列：
  触（听到领导批评） → 作意（注意力锁定） → 受（苦受标记）
  → 想（将此事件归类为"否定"） → 思（发动情绪反应）

心所链：
  触 → 受(苦) → 想("否定我价值") → 思 → 瞋/忿/恨

认知判定：
  核心错误：想心所的"取相"环节 — 将"对某件事的不同意见"
  错误地归类（想）为"对我整个人价值的否定"

Debug 策略：
  1. 在"想"环节截断：这件事 ≠ 我这人
  2. 引入善心所：无痴（如理思维：领导的评价 ≠ 客观价值）
  3. 将颠倒知降级为犹豫识："我刚才的认知对吗？" → 引入比量分析
```

### 场景 B：育儿耐心溃败

```
触发序列：
  触（听到孩子哭闹） → 作意 → 受（苦受）
  → 想（"他在故意气我"） → 思 → 瞋/忿/恼

心所链：
  触 → 受(苦) → 想(错误归因："故意") → 思 → 瞋 → 忿 → 恼

认知判定：
  核心错误：想心所的"取相"环节 — 将11个月婴儿的生理需求表达
  错误地归类为"有意图的挑衅"
  这是一个颠倒知：婴儿尚无"故意气人"的心智能力（这是经量部/唯识都承认的认知事实）

Debug 策略：
  1. 引入正比量：
     ⟨11个月婴儿的行为⟩，应是⟨生理需求表达⟩，以⟨尚未发展出故意挑衅的认知能力⟩故。
  2. 激活善心所：无瞋（慈心）、不害（不忍伤害）
  3. 观"受"的无常性：苦受也是刹那生灭的——这是南传观禅的实操切入点
```

### 场景 C：两性得失计较

```
触发序列：
  触（感觉被忽视） → 作意 → 受（苦受）
  → 想（"她不在乎我"） → 思 → 贪（执著于回报） → 嫉/忿

心所链：
  触 → 受(苦) → 想("不在乎") → 思 → 贪(执著) → 嫉 → 忿

认知判定：
  核心错误：贪心所（执著于对等回报）+ 想心所的错误分类
  "在乎"的自性 ≠ "应得对等回报"
  → 这是同体相属的伪判定："应得回报"不是"在乎"的同体相属

Debug 策略：
  1. 应成式校验（摄类学 §6）：
     ⟨我的在乎⟩，应是⟨应得对等回报⟩，以⟨我在乎她⟩故。
     → 不周遍。
  2. 善心所置换：无贪（将"在乎"还原为不执著的关怀）+ 行舍（心平等）
  3. 观"受"的三相（南传观禅）：苦受是无常、苦、无我的
```
