## §1 总框架：三学 → 七清净 → 十六观智

### 宏观递进

```
三学（戒·定·慧）
    │
    ▼
七清净
    │
    ▼
十六观智
```

### 完整对应表

| 三学 | 七清净 | 十六观智 | 实修关键 |
|------|--------|---------|---------|
| **戒** | ① 戒清净 | — | 五戒/八戒/根律仪 |
| **定** | ② 心清净 | — | 近行定 or 刹那定 |
| **慧** | ③ 见清净 | 1. 名色分别智 | 首次粉碎"我"的邪见 |
| | ④ 度疑清净 | 2. 缘摄受智 | 照见因果链 |
| | ⑤ 道非道智见清净 | 3. 三法印智 / 4. 生灭随观智（少分） | 识别并越过观染 |
| | ⑥ 行道智见清净 | 4(满分)-12. 九种观智 | 毗婆舍那核心地带 |
| | ⑦ 智见清净 | 13-16. 种姓·道·果·返照 | 出世间——证果 |

### 类比映射

```
三学        七清净           十六观智
────────────────────────────────────
戒   = 环境配置（确保 Debug 环境安全稳定）
定   = 编译通过（心能稳定运行，不崩溃/不卡顿）
慧   = 逐步深入的反编译 + 实时调试
      · 见清净 = 识别变量类型（物质 vs 认知）
      · 度疑清净 = 追踪调用栈（因果链）
      · 道非道 = 识别伪 Bug（不是真正的错误）
      · 行道智 = 核心 Debug 过程（逐层深入）
      · 智见清净 = 修复 Bug（正对治烦恼）
```

## §2 两条修行路线

### 止乘行者（Samatha-yānika）

```
路径：先修止（得禅那）→ 出定后观照诸法三相

  证得安止定（appanā-samādhi / 禅那）
        │
        ▼
  出定，以禅那淬炼过的专注力
  转向名色观察
        │
        ▼
  依次修十六观智

特点：定力深厚，观智推进稳定。适合有大块时间专修者。
```

### 纯观行者（Suddha-vipassanā-yānika）

```
路径：直接修观，不先证禅那

  以刹那定（khaṇika-samādhi）为基础——
  在每个观照的刹那，心与所缘一起生灭，
  这种刹那刹那的专注本身就是足够的定力。
        │
        ▼
  从四界分别或色心辨别起步
        │
        ▼
  依次修十六观智

特点：不需要深度禅定基础，可以在日常生活中操作。
适合碎片化时间、高强度工作 + 育儿的在家修行者。
```

### zilan 的路线选择

基于 SKILL.md 用户画像中的"认知带宽受限、时间碎片化"，孜澜的日常实践天然倾向于**纯观路线**。但这不排除在有条件时修止——两条路线可以互补。

## §3 十六观智详解（聚焦前四智：日常可用）

后十二观的深度需要在止观双运的定中才能稳定体证。前四智在日常中即可初步触达，是"实时 Debug 心识流"的直接工具。

### 第一智：名色分别智（Nāmarūpa-pariccheda-ñāṇa）

```
目标：辨别"名法"（认知/心理现象）与"色法"（物质现象），
      如实了知这堆名色中没有"我"。

操作：
  观察当下任何一个体验：
    · 色法部分：身体的感受、姿势、呼吸、温度……
    · 名法部分：知道这些感受的那个"知道"、情绪、念头……

  关键：区分"所观"与"能观"——
    "肩膀的紧张"是色法。
    "知道肩膀紧张"是名法。
    这两者都是被观察的对象，没有一个是"我"。

日常应用：
  育儿溃败场景——
    · 色法：孩子的哭声（声波）、身体的紧绷（触觉）、心跳加速
    · 名法：愤怒的感受、想要制止的念头、"他在故意气我"的念头
    · 正观：这些都是被观察的名色法——"愤怒"不是我，"念头"不是我
```

### 第二智：缘摄受智（Paccaya-pariggaha-ñāṇa）

```
目标：照见因果链——名色法不是随机生起的，而是因缘和合而生。

操作：
  观察因果的相续：
    · 听到哭声（触 → 六识中的耳识 + 意识）→ 缘
    · 苦受生起（受心所）→ 缘
    · 愤怒生起（瞋心所）→ 果
    · 身体紧绷（色法变化）→ 果
    · 更多的苦受……→ 新的缘

  关键：不是找"谁"在推动因果，而是看到因果本身的自运作。
    "动念是因，动作为果，不见有人我作。"

日常应用：
  职场否定场景——
    听到批评 → 受心所标记为苦 → 想心所归类为"否定"
    → 瞋心所生起 → 思心所发动情绪反应 → 身体产生紧张 → 更多苦受……

    这不是"我"的反应——这是一条自动化的因果链。
    看到了这条链 = 在缘摄受智中。
```

### 第三智：三法印智（Sammasana-ñāṇa）

```
目标：以无常、苦、无我三相印证一切名色法的共性。

操作：
  对所观察的任何名色法，以三相审察：

  无常（anicca）：
    "刚才的愤怒还在吗？——不在了。"
    "上一秒的念头还在吗？——不在了。"
    一切名色法都在刹那生灭。没有一个停留超过一刹那。

  苦（dukkha）：
    "被生灭逼迫——没有安稳。"
    凡是无常的，都被生灭所逼迫，被逼迫即是苦。

  无我（anattā）：
    "如果能控制，请让愤怒不要生起。——做不到。"
    "如果能控制，请让平静持续下去。——做不到。"
    凡是无常、苦的，不可能是"我"或"我的"。
    "我"的定义应该包含自主控制——但我们对名色法没有自主控制。

日常应用：
  两性计较场景——
    贪求回报的念头 → 生起了 → 持续了 N 秒 → 灭去了
    → 生起了另一个念头："她根本不在乎" → 持续了 M 秒 → 灭去了
    → 身体紧张生起 → 持续 → 灭去……

    三相印证：每个心理事件都是无常的、被生灭逼迫的（苦）、不可控的（无我）。
    "我正在计较" → 这只是一系列名色法的相续，没有"我"在计较。
```

### 第四智：生灭随观智（Udayabbaya-ñāṇa）

```
目标：刹那刹那地直接观见名色法的生起与灭去。

操作（初阶·日常可用）：
  选择一个明显的名色现象（如呼吸、身体的紧张、或某个情绪），
  标记它的"生起"和"灭去"：
    · 吸气……"生"
    · 呼气……"灭"
    · 愤怒的感受……"生"
    · 愤怒的消散……"灭"
    · 想说话反驳的冲动……"生"
    · 冲动被觉察到……"灭"

关键警告——「十种观染」（vipassanūpakkilesa）：
  在生灭随观智阶段，可能出现：
    · 异光（看见光）· 喜乐（异常愉悦）· 轻安（极度轻松）
    · 胜解（过度自信）· 策励（过度精进）· 智（误以为已证果）……
  
  这些是"观染"——不是真正的成就，而是修行路上的陷阱。
  识别方式：对这些现象也要以三相观察——
  "光是无常的吗？是的。光是可执著的吗？不能。"
  → 这恰好是"道非道智见清净"的关卡。

日常应用的关键限度：
  这是十六观智中"光明面"与"黑暗面"的分水岭。
  前四智可以在日常中操作（观心的生灭是随时可以做的），
  但在碎片化环境中的深度有限——需要专修时间才能进入坏灭随观智及之后。
```

## §4 三大长间隙的观禅应用

### 场景 A：职场否定

```
观禅 Debug 流程：

① 名色分别（第一智）：
   "领导的批评" = 声波（色法）
   "我的紧张" = 身体感受（色法）
   "知道被批评了" = 意识（名法）
   "觉得被否定了" = 想心所的归类动作（名法）
   → 这些都是名色法，没有"我"在其中。

② 缘摄受（第二智）：
   声波（缘）→ 耳识（识）→ 苦受（受）→ 归类为"否定"（想）→ 瞋生起（心所）
   → 没有"我"在推动这条链，是自动运行的。

③ 三相印证（第三智）：
   "这种难受的感觉，会永远持续吗？" → 不，它正在变化。
   "我能控制它不出现吗？" → 不——控制本身就是妄想。
   → 它是无常的、被生灭逼迫的（苦）、不可控的（无我）。

④ 回到当下的名色：
   吸气……"生"；呼气……"灭"。
   在呼吸的一进一出中，刚才的整个情绪风暴已经过去了。
```

### 场景 B：育儿耐心溃败

```
观禅 Debug 流程：

① 名色分别（第一智）：
   孩子的哭声 = 声波（色法）
   身体紧绷、心跳加速 = 色法
   愤怒 = 瞋心所（名法）
   "他在故意气我" = 想心所的错误归因（名法）
   → 关键在于：区分"事实"（名色法的生灭）与"叙事"（想心所的二次加工）

② 缘摄受（第二智）：
   哭声（缘）→ 耳识 + 意识 → 苦受标记 → 想心所归类（"故意"）→
   瞋生起 → 身体反应 → 更强烈的苦受 → 更多瞋……
   → 这是一条反馈循环。看到它 = 第一个裂隙。

③ 在三相中截断循环：
   标记：瞋生起……"生、灭"。
   标记：下一个瞋的念头……"生、灭"。
   标记：想骂人的冲动……"生、灭"。
   → 标记本身就是正念——标记的刹那，你已经不在瞋中了。
   → 这不是"压抑情绪"——而是在当下如实观察到它的无常性。
```

### 场景 C：两性得失计较

```
观禅 Debug 流程：

① 名色分别（第一智）：
   "她不在乎我" = 想心所（名法）
   "我需要被在乎" = 贪心所（名法）
   "为什么付出没有回报" = 伺心所（名法）的反复审查
   胸口的闷感 = 色法
   → 这个"我"只是一堆名色法的暂聚。没有一个实体的"受伤害的我"。

② 缘摄受（第二智）：
   她的某个行为 → 眼识/耳识 → 意识组织为"忽视"的叙事 →
   贪心所（执著于回报）→ 嫉心所 → 忿心所 →
   思心所发动"冷战或争吵"的业……
   → 这条链从"她的行为"到"我的反应"，中间经过了意识的重重组装。

③ 三相印证（第三智）：
   "我在乎她"这个念头 → 生起 → 灭去
   "她不在乎我"这个念头 → 生起 → 灭去
   "这不公平"这个念头 → 生起 → 灭去
   → 每一个念头都是无常、不可控的
   → "计较"本身就是一系列无常的念头——它不是"我"，只是一个过程。
```

## §5 与应成论式 Debug 的互补关系

### 两个工具的分工

```
应成论式 Debug（摄类学 §6）：
  · 操作对象：命题（"我的工作没有价值"）
  · 操作方式：逻辑分析（因不成立？不周遍？）
  · 输出：结论——这个命题在逻辑上不成立
  · 局限：逻辑上知道 ≠ 情绪上解脱

南传观禅 Debug（本指南）：
  · 操作对象：名色法（身体感受、情绪、念头的直接体验）
  · 操作方式：直观观察（生灭·三相）
  · 输出：体验——直接体证名色的无常、苦、无我
  · 局限：缺少对命题结构的逻辑分析
```

### 联合使用协议

```
在三大长间隙中，两个工具是互补的，使用顺序如下：

Phase 1: 应成论式 — 逻辑拆解
  先以应成论式检验"自动生成的隐含论式"是否成立。
  这一步给你一个逻辑上的"理由"来暂停反应——
  "等等，这个推理不周遍。"

Phase 2: 心类学 — 认知状态判断
  当前这一刻的认知属于七种认知中的哪一种？
  如果是颠倒知/犹豫识/伺察意 → 不该依之行动。

Phase 3: 南传观禅 — 直接体验
  在逻辑暂停的间隙，直接观察名色法的生灭：
    · 名色分别（这在逻辑暂停后自然变得更清晰）
    · 三相印证（每个情绪事件的无常性直接可见）
    · 生灭随观（标记念头的生起和灭去）

Phase 4: 回到应成论式（可选）
  观禅让情绪风暴平息后，可以用应成论式回看刚才的论式：
  "刚才那个论式，现在看还是'因不成'吗？"
  → 在平静中，逻辑结论与体验认识合一。
```

### 与中观见地的统一

```
南传观禅的三相 → 中观的无自性：

  观禅：直接观察到"没有任何一个名色法能停留超过一刹那"
  → 这就是"无常"的直接体证

  中观：如果没有任何一法能停留一刹那，
        那么没有任何一法有独立自存的自性。
        → 这就是"无自性 = 空性"的逻辑证据

  观禅提供了体验层面的"无常即无我"，
  中观提供了逻辑层面的"无常即无自性"。
  两者指向同一个事实：没有常一自在的"我"。
```
