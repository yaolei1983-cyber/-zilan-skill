## §1 性相与所表 —— 类型定义系统

### 概念映射

```
性相（mtshan nyid / definition）  = type definition / interface
所表（mtshon bya / definiendum）  = instance / implementation
名相（ming / term）               = variable name
```

### 定义

**性相**是"能立"——确立一个概念之所以为该概念的充分必要条件。
**所表**是"所立"——被性相所定义的那个概念本身。

### 三要素校验

一个合法的性相-所表关系必须同时满足三个条件：

| # | 校验条件 | 逻辑形式 | 含义 |
|---|---------|---------|------|
| ① | **性相周遍所表** | ∀x: 性相(x) → 所表(x) | 凡是具备性相者，必然是所表 |
| ② | **性相唯在所表上成立** | ¬∃x: 性相(x) ∧ ¬所表(x) | 不存在"有性相但不是所表"的例外 |
| ③ | **性相本身是实有法** | — | 性相必须对应真实存在的特征，不能是虚构 |

### 示例：以"无常"为例

```
名相：无常
性相：刹那生灭者（剎那性）
所表：一切有为法

校验：
  ① 凡是刹那生灭者，必然是无常 → ✓ 周遍
  ② 不存在"刹那生灭但不是无常"者 → ✓ 唯在所表上成立
  ③ "刹那生灭"是真实可观察的特征 → ✓ 实有法
```

### 常见错误

| 错误类型 | 示例 | 违反条件 |
|---------|------|---------|
| 性相过宽 | "能盛水者是瓶之性相"（湖也能盛水） | 违② |
| 性相过窄 | "能装墨水的陶瓷容器是瓶之性相"（排除花瓶） | 违① |
| 循环定义 | "无常是常法之对立法"（未给出正向特征） | 违③ |

### 阿含实例：《杂阿含经》卷一·第1经

> 世尊告诸比丘：「当观色无常。如是观者，则为正观。」

此处佛陀给出：
- **所表**：色蕴
- **所立法**：无常
- **隐含性相**：可被观见其生灭变化者 → 即"剎那生灭"的实操版本——"可观其无常"本身成为性相的运作定义

## §2 总与别 —— 继承/子类关系

### 概念映射

```
总（spyi / universal）   = base class / supertype
别（bye brag / particular）= subclass / subtype
```

### 定义

- **总**：在多个"别"上共通的类概念。总周遍于别。
- **别**：在"总"之下，与同总中的其他别相区分的子类。

### 判定"别"的三因

要确认 X 是 Y 的"别"（即 X 继承自 Y，X 是 Y 的子类），必须同时满足：

| # | 条件 | 含义 | 类比 |
|---|------|------|------|
| ① | **是总** | X 必须是 Y 的实例（X ⊆ Y） | `X instanceof Y` |
| ② | **与总中余者异** | X 与 Y 下的其他别类必须不同 | `X ≠ Z (where Z extends Y, Z ≠ X)` |
| ③ | **与总中余者非自性一** | X 与 Y 下的其他别类不是同一个实体 | `typeof X !== typeof Z` |

### 示例：色法的总别层级

```
一切法 (总)
├── 常法 (别)
└── 无常法 / 物 (别)
    ├── 色法 (别)
    │   ├── 外色 (别)：色处、声处、香处、味处、触处
    │   └── 内色 (别)：眼根、耳根、鼻根、舌根、身根
    ├── 心法 (别)
    └── 不相应行法 (别)
```

验证"色法是物的别"：
- ① 色法是物 → ✓（色法有为，有生灭）
- ② 色法与心法、不相应行法不同 → ✓
- ③ 色法与心法不是同一实体 → ✓

### 与"一与异"的关系

- **一与异**处理的是**两个个体**之间的关系：`a == b` 还是 `a != b`
- **总与别**处理的是**类与子类**之间的关系：`B extends A`
- "总与别"是纵向层级，"一与异"是横向比较

## §3 一与异 —— 同一性判别

> 本节整合自 SKILL.md 已有内容，补充完整的三重判别框架。

### 概念映射

```
一（gcig / identity）= 同一引用地址（两个指针指向同一个对象）
异（tha dad / difference）= 不同引用地址
```

### 三层次的"一"

| 层次 | 藏文 | 含义 | 类比 |
|------|------|------|------|
| **自相一** | rang gi mtshan nyid kyis gcig | 实体层面完全同一 | `a === b` — 同一个对象 |
| **返体一** | ldog pa gcig | 概念层面分类相同 | `typeof a === typeof b` — 同一类型 |
| **总聚一** | tshogs pa gcig | 集合层面等同 | `Set(a) === Set(b)` — 同一集合 |

### 三层次的"异"

| 层次 | 含义 | 示例 |
|------|------|------|
| **自相异** | 实体不同 | 瓶 ≠ 柱（两个独立物体） |
| **返体异** | 概念分类不同 | 瓶是色法，虚空是常法 |
| **总聚异** | 集合结构不同 | 森林 ≠ 单棵树 |

### 关键推理法则

```
若 X 与 Y 是自相一 → 则必是返体一（反之不必然）
若 X 与 Y 是返体异 → 则必是自相异（反之不必然）
```

**陷阱**：两个事物可以是"返体一、自相异"——例如金瓶和银瓶都是"瓶"（返体一），但不是同一个物体（自相异）。

### 判定协议

```
比较 X 与 Y 的同一性：

1. 自相层面：X 和 Y 是同一个实体吗？
   → 是 → 自相一，终止（所有更高层自动为一）
   → 否 → 自相异，继续

2. 返体层面：X 和 Y 属于同一个概念类别吗？
   → 是 → 返体一、自相异（同名不同物）
   → 否 → 返体异，继续

3. 总聚层面：X 和 Y 是同一个集合的不同呈现吗？
   → 是 → 总聚一、返体异（同一集合的不同子集）
   → 否 → 总聚异（彻底无关）
```

## §4 相违与相关 —— 对立与从属

> 本节整合自 SKILL.md 已有内容，补充完整分类体系。

### 相违（'gal ba / opposition）—— 互斥关系

两个法不能同时在同一有法上成立。

| 类型 | 藏文 | 含义 | 类比 | 示例 |
|------|------|------|------|------|
| **不并存相违** | lhan cig mi gnas pa'i 'gal ba | 两者不可共存于同一时空 | 互斥锁（mutex） | 明与暗、冷与热 |
| **互排相违** | phan tshun spangs pa'i 'gal ba | 逻辑上非此即彼 | `enum` 的两个 variant | 常与无常、所作与非所作 |

**判别特征**：
- 不并存相违：**物理层面**的排斥。一方存在，另一方即消失。
- 互排相违：**逻辑层面**的排斥。一个事物的自性即排除另一事物。

### 相关（'brel ba / relation）—— 从属关系

两个法之间存在不可分离的逻辑关联。

| 类型 | 藏文 | 含义 | 类比 | 示例 |
|------|------|------|------|------|
| **同体相属** | bdag gcig 'brel | 自性上的从属 | 类型系统里的子类型关系 | 无忧树 → 树（自性因） |
| **彼生相属** | de byung 'brel | 因果上的从属 | 构造函数的输入输出 | 烟 → 火（果性因） |

**关键区别**：
- 同体相属：一个概念包含了另一个概念（分析命题）
- 彼生相属：一个现象导致另一个现象（综合命题）

### 判定协议

```
检查 X 与 Y 的关系：

1. X 与 Y 能在同一有法上共存吗？
   → 否 → 相违
        → 是空间层面的排斥？ → 不并存相违
        → 是逻辑层面的排斥？ → 互排相违
   → 是 → 继续

2. X 与 Y 之间存在必然关联吗？
   → 是 → 相关
        → X 的自性包含 Y？ → 同体相属（X ⊢ Y）
        → X 的产生导致 Y？ → 彼生相属（X → Y）
   → 否 → 既非相违也非相关（偶然共存）
```

## §5 四句逻辑与周遍八门 —— 外延关系全集

### 四句逻辑（mu bzhi / tetralemma）—— 交集判断

> 基础版本，已存在于 SKILL.md。

对两个概念 A 与 B 的实例范围做穷尽交叉分析：

| 句 | 逻辑形式 | 含义 | 示例：A="无常" B="所作" |
|----|---------|------|-------------------------|
| 第一句 | A ∧ B | 既是 A 又是 B | 瓶（既无常又所作） |
| 第二句 | ¬A ∧ B | 不是 A 但是 B | —（所作皆无常，此句空） |
| 第三句 | A ∧ ¬B | 是 A 但不是 B | —（无常皆所作？需检验） |
| 第四句 | ¬A ∧ ¬B | 既不是 A 也不是 B | 虚空（非常法，非所作） |

**使用场景**：
- 检查两个概念的逻辑关系是否与声称的一致
- 对齐不同佛教派别对同一法相的不同立场（四句对齐）

### 周遍八门（khyab pa sgo brgyad）—— 外延关系全集

> 四句逻辑的自然推广。四句覆盖了交集关系，周遍八门覆盖了**包含关系**。

| # | 名称 | 逻辑形式 | 含义 | 对应论证角色 |
|---|------|---------|------|-------------|
| 1 | **后遍** | A ⊆ B | 凡是 A 必然是 B | 随遍（同品定有） |
| 2 | **下遍** | B ⊆ A | 凡是 B 必然是 A | 后遍的逆 |
| 3 | **遣遍** | ¬A ⊆ ¬B | 凡不是 A 必然不是 B | 逆遍（异品遍无的另一种表述） |
| 4 | **违遍** | ¬B ⊆ ¬A | 凡不是 B 必然不是 A | 遣遍的逆 |
| 5 | **后遍之颠倒** | ¬(A ⊆ B) | 存在 A 而不是 B | 后遍不成立 |
| 6 | **下遍之颠倒** | ¬(B ⊆ A) | 存在 B 而不是 A | 下遍不成立 |
| 7 | **遣遍之颠倒** | ¬(¬A ⊆ ¬B) | 存在不是 A 却是 B | 遣遍不成立 |
| 8 | **违遍之颠倒** | ¬(¬B ⊆ ¬A) | 存在不是 B 却是 A | 违遍不成立 |

### 与因三相的直接对应

```
因三相 → 周遍八门的实际应用

第一相（遍是宗法性）：
  因 ⊆ 有法？→ 后遍检查「凡是因的实例，必然在宗有法上成立」

第二相（同品定有性）：
  因 ∩ 同品 ≠ ∅ → 后遍的存在命题版本「同品中至少存在一个是因」

第三相（异品遍无性）：
  因 ∩ 异品 = ∅ → 遣遍「凡是非所立法者，必然非因」
```

### 与四句逻辑的关系

```
四句逻辑检查两个集合的交集分布（4 种情况）
周遍八门检查两个集合的包含关系（8 种情况）

组合使用 = 概念外延关系的完整分析能力
```

## §6 应成论式 —— 辩论协议（推理运行时）

### 定位

应成论式（thal 'gyur / prasaṅga）是摄类学逻辑推理的**实际运行环境**。前述 §1–§5 的所有工具都是静态的"语法检查器"，应成论式是连接它们的**运行时协议**——它定义了：如何接收一个命题、如何拆解其结构、如何调用各工具校验、如何返回结构化结论。

### 三要素

```
应成论式 = ⟨ 有法, 应成法, 因 ⟩

  有法（chos can / subject）     — 讨论对象        → 类比：函数参数
  应成法（thal chos / predicate） — 所欲论证的属性   → 类比：返回值类型声明
  因（rtags / reason）          — 论证依据        → 类比：函数体
```

### 格式

```
⟨有法⟩，应是⟨应成法⟩，以⟨因⟩故。
```

**示例**：

```
⟨声⟩，应是⟨无常⟩，以⟨所作性⟩故。
```

### 三种答辩（返回状态码）

| 状态码 | 含义 | 触发条件 | 对应工具 |
|--------|------|---------|---------|
| **承许**（'dod） | 论证成立 | 因在有法上成立 **且** 因周遍于应成法 | — |
| **因不成**（rtags ma grub） | 因在有法上不成立 | 因不是有法的属性 | §1（性相校验失败）、§3（一与异失配） |
| **不周遍**（khyab pa ma byung） | 因成立但不必然推出应成法 | 因与应成法之间无周遍关系 | §4（相违/相关误判）、§5（周遍破裂） |

### 三种答辩的精确判定流程

```
收到论式：⟨有法 A，应是 B，以 C 故⟩

Step 1: 因校验 — C 在 A 上成立吗？
  使用 §1（性相-所表）：A 的性相是否包含 C？
  使用 §3（一与异）：C 与 A 的返体关系是什么？
  → 成立 → 继续 Step 2
  → 不成立 → 返回「因不成」

Step 2: 周遍校验 — C 周遍于 B 吗？
  使用 §2（总与别）：C 是 B 的"别"还是"总"？
  使用 §4（相违与相关）：C 与 B 是相属关系吗？
  使用 §5（周遍八门）：C ⊆ B 成立吗？
  → 成立 → 返回「承许」
  → 不成立 → 返回「不周遍」
```

### 五种应成论式立法类型

| 类型 | 特征 | 示例 |
|------|------|------|
| **因法义各别** | 单一因、单一应成法 | ⟨声，应是所作，以是物故⟩ |
| **有法重叠** | 对有法的多重论证 | ⟨声，应是所作，以是物故⟩∧⟨声，应是无常，以是所作故⟩ |
| **后陈堆砌** | 单一因推出多重应成法 | ⟨声，应是无常、所作、物，以是有为法故⟩ |
| **因堆砌** | 多因支持单一应成法 | ⟨声，应是无常，以所作性、有生灭、是物故⟩ |
| **里层充塞** | 嵌套论证 | ⟨声，应是无常，以所作故⟩；若因不成，则续证「声是所作」 |

### 实修应用：三大长间隙的应成式 Debug

#### 场景 A：职场否定

```
触发：被领导质疑工作价值
自动生成的隐含论式：

⟨这份工作⟩，应是⟨没有价值的⟩，以⟨被领导否定⟩故。

应成式 Debug：
  Step 1 因校验：被领导否定 → 因成立（事实发生）
  Step 2 周遍校验：
    被领导否定 ⊆ 工作无价值？
    → 违遍检查：是否存在「被领导否定 但 工作有价值」的情况？
    → 存在！（领导也会犯错、否定可能是暂时的、价值参照系不应外包）
  → 结论：不周遍
  → 修正：不调整"工作有无价值"的结论，而是审视"谁的评价构成有效参照系"
```

#### 场景 B：育儿耐心溃败

```
⟨此刻的情绪爆发⟩，应是⟨合理的反应⟩，以⟨孩子执意妄为⟩故。

  Step 1 因校验：孩子执意妄为 → 因成立
  Step 2 周遍校验：
    孩子执意妄为 ⊆ 情绪爆发合理？
    → 不周遍：孩子哭闹是11个月婴儿的正常行为（同品中有"非合理反应"存在）
  → 结论：不周遍
  → 修正：拆分为两个独立命题
    (a) 孩子哭闹需要应对（承许）
    (b) 情绪爆发是有效应对方式（因不成——情绪爆发恰恰降低应对能力）
```

#### 场景 C：两性得失计较

```
⟨我的付出⟩，应是⟨应得到对等回报⟩，以⟨我在乎她⟩故。

  Step 1 因校验：我在乎她 → 因成立
  Step 2 周遍校验：
    「在乎」⊆「应得对等回报」？
    → 不周遍：在乎是单向的情感状态（同体相属：在乎的自性不包含"被回报"）
    → "应得回报"不是"在乎"的同体相属，而是额外附加的遍计所执
  → 结论：不周遍
  → 修正：将"计较"识别为遍计所执性，回调至依他起性的"在乎"本身
```

## §7 破与立 + 排他 —— 论证策略与否定语义

### 破（遮止 / dgag pa）—— 否定论证

**破**不主张任何正面的论题，而是暴露对方论题的自相矛盾。这是中观方法论的核心操作——"破而不立"。

| 类型 | 含义 | 逻辑形式 | 示例 |
|------|------|---------|------|
| **无遮遮止** | 只否定，不隐含任何肯定 | ¬P（纯否定） | "声非常"——只否定"声是常"，不隐含"声是什么" |
| **非遮遮止** | 否定后隐含了另一个肯定 | ¬P → Q（排除性否定） | "这不是桌子（这是一把椅子）"——排除"桌子"后自然指向了其他 |

**使用策略**：
- 当对方论题有逻辑缺陷时 → 使用**无遮**：只暴露矛盾，不给对方新的靶子
- 当需要在否定后给出替代方案时 → 使用**非遮**：否定 + 暗示正确方向

### 立（成立 / sgrub pa）—— 肯定论证

**立**正面建立己方的主张，必须依赖完整的论证结构：

```
能立 = 能立因 + 能立喻

能立因：满足因三相的正因
能立喻：同法喻（正例）+ 异法喻（反例）
```

### 排他（apoha / gzhan sel）—— 概念的否定语义学

**排他**是佛教认识论的核心理论：一个概念之所以有意义，不是因为它指向了什么正面的"共相"，而是因为它**排除了所有与之相违的概念**。

```
概念"瓶" ≠ 有一个正面的"瓶性"（若有，则瓶性是常是物？无法归类）
概念"瓶" = 排除一切非瓶者（排他）后，概念在认知中安立
```

**三阶段运作**：

| 阶段 | 操作 | 示例：理解"无常" |
|------|------|-----------------|
| ① 所破确定 | 确认要排除什么 | 排除"常"（不变易、无生灭） |
| ② 遮止执行 | 将排除范围施加于认知对象 | 对此物：确认其非"常" |
| ③ 安立 | 在排除后，概念在认知中确立 | "无常"作为"常法之否定"被理解 |

### 破、立、排他 三者协作

```
论证的完整生命周期：

1. 排他定义边界：概念的精确语义通过排除互排相违来确立
2. 破清除障碍：用无遮/非遮排除对方的逻辑错误
3. 立建设正见：用能立因+能立喻建立正确的观点

三者关系：
  排他 = 类型系统的语义基础（概念如何获得意义）
  破   = 纠错机制（为什么对方的推理不成立）
  立   = 构建机制（如何建立正确的推理）
```
