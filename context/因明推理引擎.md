## §1 因三相 —— 推理正确性验证器

### 定位

因三相（trairūpya / tshul gsum）是陈那确立的论证理由有效性检验标准。一个论证要成立，其"因"（理由）必须同时满足三个条件。这三个条件恰好对应摄类学工具箱中的概念工具——因三相不是新概念，而是已有工具在推理验证场景中的组合调用。

### 三相对应关系

```
因三相                  校验内容                  对应摄类学工具
────────────────────────────────────────────────────────────
第一相 · 遍是宗法性      因必须在有法上成立          §1 性相与所表 + §3 一与异
第二相 · 同品定有性      因在同品中至少存在一处       §2 总与别 + §5 周遍八门(后遍)
第三相 · 异品遍无性      因在异品中完全不存在         §5 周遍八门(遣遍)
```

### 第一相：遍是宗法性（pakṣa-dharmatā / phyogs chos）

```
定义：因必须是"有法"的属性。因的外延必须覆盖有法。

逻辑形式：有法 ⊆ 因
校验方法：
  → §1（性相与所表）：有法的性相是否包含"因"？
  → §3（一与异）：因与有法是否至少是"返体一"的关系？

示例：
  ⟨声，应是所作，以是物故⟩
  第一相校验：声是物吗？→ 是 ✓
```

### 第二相：同品定有性（anvaya-vyāpti / rjes khyab）

```
定义：因必须在"同品"（即具有所立法属性的事物）中至少存在一处。
这仅仅是一个存在命题，不是全称命题——"同品中至少有一个是因"。

逻辑形式：∃x: 同品(x) ∧ 因(x)  （不是 ∀x: 同品(x) → 因(x)）
校验方法：
  → §2（总与别）：因是所立法的"别"还是"总"？
  → §5（周遍八门）：后遍（因 ⊆ 所立法）成立吗？

示例：
  ⟨声，应是无常，以所作性故⟩
  第二相校验：同品 = 无常物。其中是否存在"所作性"？
  → 是的，瓶既是无常物又是所作性 → 存在 ✓
```

**关键：为什么是"定有"而非"遍是"？**

陈那的因明保留了不共不定因的空间：因可以只在同品的一个子集中存在，只要不同时出现在异品中。法称后来取消了这一宽松性，但这已进入学术争议层面，不在 P1 范围。

### 第三相：异品遍无性（vyatireka-vyāpti / ldog khyab）

```
定义：因必须在"异品"（即不具有所立法属性的事物）中完全不存在。
这是一个全称否定命题。

逻辑形式：∀x: 异品(x) → ¬因(x)  （全称否定）
等价形式：因 ∩ 异品 = ∅
校验方法：
  → §5（周遍八门）：遣遍（¬所立法 ⊆ ¬因）成立吗？
  → 即：凡不是所立法者，必然不是因

示例：
  ⟨声，应是无常，以所作性故⟩
  第三相校验：异品 = 常法。其中是否存在"所作性"？
  → 不，常法中无一是所作性（所作性必无常）→ 遍无 ✓
```

### 因三相完整校验协议

```
输入：论式 ⟨有法 A，应是 B，以 C 故⟩

Phase 1: 第一相校验
  Q1: C 在 A 上成立吗？（C ⊆ A？A 的性相是否包含 C？）
  → YES → 进入 Phase 2
  → NO  → 返回「因不成」

Phase 2: 第二相校验
  Q2: 同品（具有 B 属性的事物）中至少存在一处是 C 吗？
  → YES → 进入 Phase 3
  → NO  → 返回「因在同品中不存在」

Phase 3: 第三相校验
  Q3: 异品（不具有 B 属性的事物）中完全不存在 C 吗？
  → YES → 返回「因三相完全满足，因是正因」
  → NO  → 返回「因在异品中存在（不定因/相违因）」

三阶段全 PASS → 论题成立。
```

## §2 三因说 —— 推理类型分类器

### 定位

法称将陈那的因三相提炼为更简洁的**三因说**——所有有效的正因最终都可以归入三种类型之一。如果说因三相是推理的**语法校验器**（形式是否正确），三因说就是推理的**类型系统**（实质属于哪一种推理关系）。

### 核心创新

法称的关键突破在于：将陈那依赖"辩论双方共许（共许极成）"的不相离性，替换为基于**客观实在世界中的自然关系**的不相离性。这意味着：

- 陈那：因与所立法之间的必然关联依赖于辩论双方共同认可
- 法称：因与所立法之间的必然关联依赖于**事物自身的同一关系或因果关系**

这使佛教逻辑完成了从"类比推理"到"演绎逻辑"的跨越。

### 三种正因

| # | 类型 | 梵文 | 藏文 | 逻辑基础 | 比喻 |
|---|------|------|------|---------|------|
| 1 | **不可得因** | anupalabdhi-hetu | mi dmigs pa'i rtags | 认知的缺失 | "此处无瓶，瓶可得相虽已具足，而瓶不可得故" |
| 2 | **自性因** | svabhāva-hetu | rang bzhin gyi rtags | 同一关系（同体相属） | "此物是树，以彼是无忧树故" |
| 3 | **果性因** | kārya-hetu | 'bras bu'i rtags | 因果关系（彼生相属） | "彼处有火，以见烟故" |

### 不可得因 —— 否定推理

```
逻辑形式：若 P 且可感知条件具备，则应感知到 Q；
          实际未感知到 Q；故 ¬P。

关键约束：不可得因的使用需要满足"可得相已具足"——
即"如果它存在，我们就应该能感知到它"这个前提必须成立。

示例：
  "桌上的杯子不存在，因为：
   ① 如果杯子在桌上，以我的眼根现量应当能见到它（可得相具足）
   ② 实际没有见到杯子（不可得）
   ③ 故桌上无杯。"

伪不可得因（误用）：
  "抽屉里没有杯子，因为我没看见。"
  → 不成立：可得相未具足（抽屉关着，看不见内部）
```

### 自性因 —— 同一关系推理

```
逻辑形式：因（C）与所立法（B）是同体相属关系
          即 C 的自性就包含 B 的概念

对应摄类学 §4（相违与相关）中的"同体相属"

同体相属校验：
  · C 是 B 的"别"？（C extends B）→ §2 总与别
  · C 的自性中必然包含 B 的自性？→ §1 性相与所表

示例：
  ⟨此物，应是树，以是无忧树故⟩
  "无忧树"的自性定义中已经包含了"树"——
  无忧树是树的子类（别），是"同体相属"关系。
```

### 果性因 —— 因果关系推理

```
逻辑形式：因（C）是所立法（B）的果
          即 C 与 B 是彼生相属关系

对应摄类学 §4（相违与相关）中的"彼生相属"

彼生相属校验：
  · B 导致 C？（因果方向：所立法导致因，而非因导致所立法）
  · 因果的必然性成立吗？

示例：
  ⟨彼处，应有火，以见烟故⟩
  烟（C）是火（B）的果——火导致烟，故见烟可推知有火。
  因果方向：B（火）→ C（烟），推理方向：C（烟）→ B（火）

注意方向：
  因（果性因）= 果
  所立法 = 因（原因）
  果性因是用"果"来推"因"，不是反过来。
```

### 三因判定协议

```
输入：一个通过了因三相验证的正因 C

判定 C 属于哪种正因：

1. C 与所立法 B 是同体关系吗？
   → 使用 §4：C 与 B 是同体相属？C 是 B 的"别"？
   → YES → 自性因

2. C 是所立法 B 的果吗？
   → 使用 §4：C 与 B 是彼生相属？B 导致 C？
   → YES → 果性因

3. C 是"本应感知到的 B 却未感知到"吗？
   → 结论是要否定 B 的存在？
   → YES → 不可得因

三者互斥且穷尽——所有有效正因必然属于其中一种。
```

### 与实修的对应

| 三因 | 修行场景 |
|------|---------|
| **自性因** | 分析概念关系：烦恼的自性是什么？"瞋"的自性是否包含"苦受"？（同体相属） |
| **果性因** | 追溯因果链：此刻的情绪（果）是由什么因缘导致的？ |
| **不可得因** | 检查执着：我执执取的"常一自在之我"，在五蕴中可得吗？（不可得 → 无我） |
